Machine Learning (AI) Projects

A collection of small AI projects. They're just 'fun' small projects I made to acquaint myself with machine learning. Check them out below!

Bert Checker

For this project I trained a model to recognise my dog, Bert. Around this I build a simple UI and some interaction. Because the model is only trained on my dog it will only recognise my dog or dogs that look like him, or maybe a rag that's kind of the same color as him. This makes it a little hard to use the application if you don't have any of that. So besides a demo and the code there's also a video to see it work!

Doodle Game

For this project I worked with ML5 Neural Networ Image Classifying. I used a pre-trained model to make a doodle game, you get a prompt to draw a certain thing, upload an image of it and get judged by the AI. Next to that I used BrowserSpeech so if you turn the sound on you can feel extra judged. Try the demo or check out the code!

Picture Perfect

For this project I worked with the K-Nearest-Neighbor algorithm. The algorithm is trained to recognise pixelated versions of a couple of van Gogh paintings. The webcam takes in new data which gives an array and the algorithm finds the most similar (or the nearest neighbor) out of the trainingsdata. So the app can tell what van Gogh painting the current webcam image looks most like. Try it out yourself or check out the code!
